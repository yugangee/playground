### 대회 연혁 요약

한국기자협회 축구대회는 1972년 처음 개최되어 올해로 51회째를 맞는 유서 깊은 대회다. 회원 언론인의 체력 향상과 친목 도모를 목적으로 시작됐으며, 회를 거듭하며 규모가 확대됐다. 2014년 이후 서울지역 대회로 운영되고 있으며, 2020~2021년에는 코로나19로 대회가 취소된 바 있다.

## 3. 참가 자격

- 한 팀당 선수 최대 **30명**까지 구성 가능
- 선수명단 제출 시 **한국기자협회 회원**인 자
- 회원 출신 편집·보도국 간부는 출전 자격 인정
- **수습기자**는 팀당 **2명까지** 가능 (재직증명서 제출 필수)
- 같은 회사라도 지회가 분리된 경우 **지회 기준**으로 팀 구성 (연합팀 불가)
- 한 지회당 하나의 팀만 출전 가능
- 팀원은 동일한 유니폼을 착용해야 하며, 각 팀원마다 다른 등번호 배번 필수
- **사전에 등록되지 않은 선수는 출전 불가**

## 4. 경기 방법

- **토너먼트** 방식 진행
- 경기 시간: **전·후반 각 15분** (휴식 10분)
- 무승부 시 **승부차기** 진행 (연장전 없음)
- 3·4위전은 승부차기로 진행 (5명씩 차고 미결 시 서든데스)
- 선수 교체: **무제한** (교체 선수 해당 경기 재출전 가능)
- 승부차기 시 후반 종료 시점 출전 중인 선수로만 순서 작성 (후반 종료 후 교체 불가)
- FIFA 및 KFA 경기규칙 적용, 그 외 사항은 협회 사무국 결정
- 공식 VAR 미운영, 개별 촬영 영상도 불인정
- 우천 등 경기 불가 시 대회 주최 측 결정에 따름

## 7. 대회 운영 규정 요약

### 제1조 경기규정

FIFA 및 KFA 경기규칙 적용. 이외 사항은 협회 사무국 결정.

### 제2조 대회운영과 경기방식

- 토너먼트 진행, 전·후반 각 15분 (휴식 10분)
- 무승부 시 승부차기, 3·4위전은 승부차기로만 진행
- 선수 교체 무제한 (재출전 가능)
- 공식 VAR 미운영, 개별 촬영 영상 불인정

### 제3조 출전선수 준수사항

- 사전 등록·승인된 선수만 출전 가능
- 본부석에서 신분 확인 후 검인 필수 (신분증 지참)
- 최소 출전 인원: **7명 이상** (미달 시 경기 불가)
- 경기 개시 시각까지 신분 확인 7명 미만 시 **몰수패**
- 부정 선수 발견 시 **몰수패**
- 등번호는 명단 기재 번호와 동일해야 하며, 위배 시 즉시 퇴장

### 제4조 선수복장

- 팀별 유니폼·스타킹 색상 동일, 상하의 배번 필수
- 고무·우레탄 스파이크 축구화 또는 풋살화만 착용 (금속 스파이크 금지)
- 정강이 보호대 및 스타킹 필수
- 안경·시계·목걸이·반지 등 부상 유발 물품 착용 금지
- 양 팀 유니폼 색상 유사 시 어웨이팀(대진표 오른쪽)이 보조 유니폼·조끼 착용
- 등번호 1~99번 등록 가능, 대회 기간 중 변경 불가

### 제5조 벌칙규정

| 상황 | 조치 |
|------|------|
| 부정선수 기용 | 성적 무관 실격 |
| 경기 불응 (5분 이내 미출전) | 몰수패 |
| 한 경기 경고 2회 → 퇴장 | 다음 **1경기** 출전정지 |
| 다른 경기 경고 2회 누적 | 다음 **1경기** 출전정지 |
| 경고 없이 즉각 퇴장 (레드카드) | 다음 **2경기** 출전정지 |
| 심판에 욕설·신체 압력 | 퇴장 + 2경기 출전정지 |

- **경고 초기화:** 8강까지 경고 1장만 받은 상태에서 4강 진출 시 이전 경고 초기화
- 8강까지 경고 누적 또는 레드카드로 인한 출전 금지는 4강 이후에도 유효
- 판정 이의는 **주장만** 제기 가능
- 징계 사항은 당해연도 대회 한정, 대회 종료 시 자동 소멸

### 제7조 기타

- 경기 전 음주 금지
- 벤치는 등록된 감독·총무·선수만 출입, 응원단은 지정 구역 이용
- 서류·신분증 위변조 시 해당 팀 **즉각 실격**

## 8. 보험 안내

협회 차원의 단체 보험은 가입되어 있지 않으며, 대회 중 부상에 대한 책임은 협회에서 지지 않는다. 선수 개별 보험 가입이 가능하다.

## 9. 협조 사항

- 경기장 내 음주·흡연·쓰레기 투기·취사·취식 **금지**
- 꽹과리·부부젤라 등 소음 유발 장비 사용 **자제**
- 응원단 강제 동원 **금지**
- 앰뷸런스는 심정지·머리 부상 등 위급 상황 시에만 의료진 판단하에 운영
- 기타 부상은 의료진 응급조치 후 개별 이동 원칙
- 분리수거 철저

## 10. AI 중계 도입

제51회 대회부터 KT스카이라이프와 업무협약(MOU)을 체결하여 AI 중계가 도입된다. KT스카이라이프의 자체 AI 중계 솔루션 '포착'을 활용해 축구대회와 제3회 여성회원 풋살대회를 중계할 예정이다. 2025년 9월 2일 서울 한국기자협회 사무실에서 최영범 KT스카이라이프 사장과 박종현 한국기자협회 회장이 협약식을 진행했다.
