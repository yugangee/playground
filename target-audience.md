# 한국기자협회 축구대회 데모 구현 계획

## 목표
제51회 한국기자협회 서울지역 축구대회를 PLAYGROUND 플랫폼의 첫 데모 케이스로 구현.
타겟 사용자(아마추어 축구팀 총무/운영자)가 실제 대회에서 활용 가능한 수준의 데모.

## 접근 방식: 프론트엔드 중심 + 정적 데이터

속도가 최우선이므로, 백엔드 변경 없이 프론트엔드에 정적 토너먼트 데이터를 내장하여 빠르게 구현합니다.
- 토너먼트 대진표는 이미 확정된 데이터 → 정적 파일로 관리
- 실시간 스코어/참석은 추후 백엔드 연동
- 기존 인증/팀 관리 API는 그대로 활용

---

## Phase 1: 토너먼트 데이터 레이어
> 정적 데이터 파일 생성

### 1-1. 타입 정의 (`types/tournament.ts`)
- `TournamentTeam`: 팀명, 시드, 소속(언론사)
- `TournamentMatch`: 조번호, 라운드, homeTeam, awayTeam, score, status
- `TournamentRound`: 라운드명(1회전, 2회전, 8강, 4강, 결승)
- `TournamentData`: 대회 전체 구조
- `PlayerRoster`: 등번호, 이름, 소속부서, 직함, 역할(주장/감독 등)

### 1-2. 제51회 대회 데이터 (`data/kja-tournament-51.ts`)
- 50개+ 팀 목록 (시드 포함)
- 55개 매치 슬롯 (1조~55조 + 결승/3·4위전)
- 좌측 블록(54조) / 우측 블록(55조) 구조
- 서경 어벤져스 선수단 20명 로스터
- 대회 규정 요약 정보

### 1-3. 제50회 대회 결과 (`data/kja-tournament-50.ts`)
- 이전 대회 순위: 우승(YTN), 준우승(동아일보), 3위(뉴스1), 4위(서울경제)

---

## Phase 2: 토너먼트 브래킷 컴포넌트
> 싱글 엘리미네이션 대진표 시각화

### 2-1. `components/tournament/TournamentBracket.tsx`
- **좌측 블록 ↔ 우측 블록** 양쪽에서 결승으로 수렴하는 레이아웃
- 라운드별 컬럼 배치 (1회전 → 2회전 → 8강 → 4강 → 결승)
- 매치 카드: 팀명, 스코어, 승패 표시
- 연결선(커넥터): 이전 라운드 → 다음 라운드 시각적 연결
- **서경 어벤져스 하이라이트**: 자팀 경로 강조 표시
- 가로 스크롤 지원 (50개+ 팀이라 모바일에서 스크롤 필요)

### 2-2. `components/tournament/MatchCard.tsx`
- 매치 카드 UI: 홈팀 vs 어웨이팀
- 상태별 스타일: 예정(회색), 진행중(파랑), 완료(결과 표시), 승부차기(PK 표시)
- 클릭 시 매치 상세 모달

### 2-3. `components/tournament/TeamRoster.tsx`
- 선수 명단 테이블: 등번호, 이름, 소속부서, 역할
- 스태프 정보: 감독, 주장, 총무, 단장
- 유니폼 정보 표시

---

## Phase 3: 토너먼트 전용 페이지
> `/league/kja-51` 라우트

### 3-1. 메인 레이아웃
- 히어로 섹션: 대회명, 일시, 장소, 참가팀 수
- 네비게이션 탭:
  1. **대진표** — 풀 브래킷 뷰
  2. **경기 일정** — 날짜별 경기 목록
  3. **서경 어벤져스** — 팀 로스터 + 경기 일정
  4. **이전 대회** — 50회 결과 + 역대 기록
  5. **대회 규정** — 주요 규정 요약

### 3-2. 대진표 탭
- Phase 2의 `TournamentBracket` 컴포넌트 렌더링
- 시드 팀 하이라이트 (1~4시드)
- 라운드 필터 (전체 / 1회전 / 2회전 / 8강~)

### 3-3. 경기 일정 탭
- 대회 일정이 확정되면 날짜별 경기 표시
- 현재는 "일정 추후 공지" 상태 표시
- 각 매치 클릭 → 상세 정보 (장소, 시간, 규정)

### 3-4. 서경 어벤져스 탭
- 팀 기본 정보 (유니폼, 스태프)
- 선수 명단 20명 (등번호, 이름, 부서)
- 부서별 분포 차트 (Recharts 활용)
- 대회 내 서경의 대진 경로 (38조부터)

### 3-5. 이전 대회 탭
- 50회 대회 결과 (1~4위)
- 역대 대회 요약 (가능한 범위)

### 3-6. 대회 규정 탭
- 경기 방식, 출전 자격, 벌칙 규정 요약
- 아코디언 UI로 항목별 펼침

## Phase 5: 경기 결과 & 스코어 입력 (데모용)
> 프론트엔드 로컬 상태로 스코어 입력 기능

### 5-1. 스코어 입력 모달
- 관리자(서경 어벤져스 총무)가 매치 결과 입력
- 홈/어웨이 스코어, 승부차기 여부
- 로컬 상태 업데이트 → 브래킷 자동 반영
- (추후 백엔드 API 연동 가능)

### 5-2. 자동 대진 진행
- 완료된 매치의 승자가 다음 라운드에 자동 배치
- 승부차기 결과도 반영
